# Agents

## Monitoring Agent
Detect anomalies from:
- CPU
- Memory
- Error Rate
- Latency

## Change Intelligence Agent
Correlate incidents with:
- Deployments
- Config changes

## Diagnostic Agent
Perform root cause analysis.

## Knowledge Agent
Search:
- Runbooks
- Documentation
- Historical fixes

## Incident Time Machine Agent
Find similar incidents.
Rank successful remediations.

## Remediation Agent
Actions:
- Restart Service
- Restart Pod
- Scale Deployment
- Rollback Deployment
- Clear Cache

## Validation Agent
Verify service recovery.

## Communication Agent
Generate:
- Engineer Summary
- Manager Summary
- Executive Summary

Workflow:

Monitoring
↓
Change Intelligence
↓
Diagnostic
↓
Knowledge
↓
Incident Time Machine
↓
Remediation
↓
Validation
↓
Communication
